1. 阅读 `../README.md` 了解项目概述
2. 参考 `../QUICK_START.md` 进行快速部署
3. 查看 `architecture_diagram.md` 理解系统架构
4. 根据需要阅读其他详细文档

4. `COMPREHENSIVE_ARCHITECTURE_UPGRADE.md` - 性能监控与升级
